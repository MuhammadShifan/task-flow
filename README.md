# Task Flow

Welcome to **Task Flow**! This is a clean and efficient web application designed to help users manage their daily tasks and improve productivity.

### Live Demo
Check out the live project here: [Task Flow App](https://task-flow-mds.netlify.app/)


### Features
* **Task Management:** Easily add, track, and manage your daily tasks.
* **Interactive UI:** Smooth and user-friendly interface for organizing your workflow.
* **Dynamic Updates:** Real-time adding and removing of tasks using JavaScript DOM manipulation.
* **Responsive Design:** Optimized for both desktop and mobile devices.


### Technologies Used
* **HTML5:** For the structural layout.
* **CSS3:** For styling and creating an engaging user interface.
* **JavaScript:** For application logic, event handling, and dynamic content updates.
